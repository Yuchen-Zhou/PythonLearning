# Django Tutorial
Python下有许多不同的Web框架，Django是重量级选手中最具代表性的一位。许多网站和App都是基于Django的。例如：
- Instagram
- 豆瓣
- 知乎

Django是一个开放源代码的Web应用框架，由Python写成
Django采用了MVT的软件设计模式，即模型（model）、视图（View）和模版（Template）
